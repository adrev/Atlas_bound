# Map Art Generation Prompts

Use these prompts with an image generation AI (Midjourney, DALL-E, Stable Diffusion, etc.)
to create top-down battle map art for the D&D VTT.

**Important**: Add these suffixes to every prompt for consistency:
- `top-down view, battle map, fantasy RPG, grid-ready, no text, no UI, high detail, digital painting`
- For the aspect ratio, use the grid dimensions listed (e.g. 30x30 = square, 40x20 = 2:1 landscape)

Once generated, save images as `.jpg` files in:
`client/public/maps/` (e.g., `goblin-camp.jpg`)
